Accessibility toolkit

A11y Audit Panel

A simple bookmark you can click on any page to open a visual accessibility audit panel. Built for QA testers, designers, and developers.

Drag to bookmarks bar Copy bookmarklet
If drag-and-drop does not work, copy the bookmarklet below and paste it into the URL or location field of a new bookmark.

What it gives you

Headings, landmarks, alt text, labels, hidden content, tab order, contrast, focus, ARIA, and form checks.
Works on any page without installing a full extension.
One bookmark stays the same while you keep shipping updates.
Public toolkit page for testers. Developer files are below.

How to use it

  1. Show your browser bookmarks bar.
  2. Drag the button above into the bookmarks bar.
  3. Open any site or local page you want to test.
  4. Click the bookmark to open the audit panel.

Copy fallback

If your browser blocks dragging, copy this value and paste it into a new bookmark’s URL/location field.
