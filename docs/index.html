Accessibility toolkit

Tester tools in one place

Drag a bookmark into your browser and use the A11y Audit Panel or the A11y UX Widget on any page. No extension needed.

If drag-and-drop does not work, copy the bookmarklet below and paste it into the URL or location field of a new bookmark.

What you get

Audit panel for accessibility testing and issue spotting.
UX widget for contrast, fonts, motion, cursor, and reading support.
One public toolkit page, one repo, shared updates.
Portfolio-backed scope: shared accessibility modules, workflows, docs, and research.

How to use

  1. Show your browser bookmarks bar.
  2. Drag one of the buttons above into the bookmarks bar.
  3. Open any site or local page you want to test.
  4. Click the bookmark to open that tool.
